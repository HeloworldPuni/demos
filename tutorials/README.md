## üìñ Terminology Guide

**New to crypto? Here's what terms mean**:

- **Blockchain** - A public ledger that records all transactions
- **Smart Contract** - Code that runs on the blockchain automatically
- **Base** - A blockchain network by Coinbase (fast and cheap)
- **Testnet** - A practice blockchain with fake money (for testing)
- **Mainnet** - The real blockchain with real money
- **Wallet** - Like a bank account, but you control the private keys
- **Private Key** - Password to your wallet (NEVER share!)
- **Gas** - Fee to run transactions on blockchain
- **USDC** - A stablecoin (cryptocurrency = $1 USD)
- **API Key** - Password to use a service's features
- **Deploy** - Upload and activate your contracts/website
